My top 5 favourite extensions for Visual Studio Code.

July, 2021

1.Live Server by RitwickDey.

It is simple to use but the most powerful extension for web developers who write lots of code in Html, CSS and JS, this watches the changes in the files and when files are saved, it reloads the browser, and changes are reflected. This prevents the user from reloading the browser after saving changes.

How to install this extension:- Open VScode -> on the left navigate to extensions -> search for "Live Server".

How to configure and use this extension:- here

2.Prettier by Prettier.

It is an auto formatter that automatically formats code on saving.it supports formatting for HTML, CSS, JS, JSON ,TS etc. It makes code more readable and easy to understand. It is very simple to configure and use.

How to install this extension:- Open VScode -> on the left navigate to extensions -> search for "Prettier".

How to configure and use this extension:- here

3.Auto Rename Tag by Jun Han.

It is a very useful extension when it comes to renaming an HTML tag, just by renaming the opening tag it automatically modifies the closing tag if it is there. It also works opposite i.e changing the closing tag automatically updates the opening tag. It is very simple to configure and use.

How to install this extension:- Open VScode -> on the left navigate to extensions -> search for "Auto Rename Tag".

How to configure and use this extension:- here

4.TODO Highlight by Wayou Liu.

It is a must-have extension for the users who prefer to write todos and comments for fixing or implement things later, it simply highlights the keywords TODO: and FIXME: with eye-catchy colours so when we see it it reminds the task. Other than TODO: and FIXME: we can add custom words with custom text and colours.

How to install this extension:- Open VScode -> on the left navigate to extensions -> search for "TODO Highlight".

How to configure and use this extension:- here

5.vscode-icons by VScode Icons Team.

It is an extension with a set of icons for files and folders, making it easy to distinguish between different types of files and folders easily just by looking at an icon.

How to install this extension:- Open VScode -> on the left navigate to extensions -> search for "vscode-icons".

How to configure and use this extension:-here
